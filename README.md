# ☑️ TodoSleep_X4

Xteink X4 기기의 슬립화면을 나만의 To-Do 리스트로 꾸미고 무선으로 전송할 수 있는 데스크톱 유틸리티 프로그램입니다.

![스크린샷](assets/screenshot.PNG)

---

## ⚠️ 필수 사전 작업

이 프로그램을 사용하시려면 다음 조건이 갖춰져야 합니다.

1. **크로스포인트 펌웨어 설치**: 본 프로그램은 크로스포인트 펌웨어 환경을 기준으로 제작되었습니다. (순정 펌웨어에서의 작동 여부는 확인되지 않았습니다.)
2. **동일한 네트워크 연결**: PC와 X4 기기가 같은 와이파이, 혹은 같은 공유기 네트워크에 연결되어 있어야 합니다. (PC는 랜선, X4는 와이파이 연결도 무방합니다.)
3. **`sleep` 폴더 생성**: X4 기기의 SD 카드 최상단에 `sleep` 이라는 이름의 폴더를 생성해 주세요. 
   * **Tip:** 폴더 안에 다른 슬립화면 이미지가 있다면 화면이 랜덤으로 출력됩니다. 항상 할 일 목록만 띄우고 싶으시다면 다른 이미지들은 지우고 빈 폴더 상태로 시작해 주세요.

## ✨ 주요 기능

### 1. X4 기기 무선 연결
PC에서 작성한 To-Do 이미지를 선 연결 없이 X4로 바로 전송합니다.

* 크로스포인트 홈 화면 ➔ **[파일 전송]** ➔ 가장 상단의 **[네트워크 연결]** 메뉴에서 IP 주소를 확인합니다. (예: `192.100.0.00`)
* X4 화면에 QR 코드가 떠 있는 상태를 유지한 채, 프로그램의 **[X4 IP 주소]** 칸에 해당 숫자를 입력하고 연결을 확인합니다.
* 전송을 누르면 할 일 목록 이미지가 `sleep` 폴더에 `todo.bmp` 파일로 덮어쓰기 되어, 별도로 이전 이미지를 삭제하지 않아도 슬립화면이 자동으로 갱신됩니다.
* **설정 저장:** 한 번 입력한 IP 주소는 프로그램과 같은 폴더에 생성되는 `settings.json`에 로컬로 안전하게 저장되며, 다음 실행 시에도 그대로 유지됩니다.

### 2. 커스텀 폰트 설정
슬립화면의 텍스트 스타일을 내 취향에 맞게 꾸밀 수 있습니다.

* 기본적으로 '맑은 고딕' 폰트가 사용됩니다.
* 개인이 소장한 `.ttf` 폰트 파일을 업로드하여 적용할 수 있으며, 글씨 크기도 세밀하게 조절 가능합니다.
* **설정 저장:** 업로드한 폰트는 `custom_font.ttf`라는 이름으로 복사되어, 프로그램을 다시 실행해도 설정한 폰트가 계속 유지됩니다.

### 3. To-Do 리스트 관리
직관적인 인터페이스로 할 일을 추가하고 관리하세요.

* **날짜 지정:** 입력창 왼쪽의 **[날짜]** 버튼을 누르면 나타나는 캘린더 창에서 특정일이나 기간을 선택할 수 있습니다.
* **상세 관리:** 할 일 목록 좌측의 아이콘을 통해 다양한 관리가 가능합니다.
  * 📌 **상단 고정:** 정렬 방식에 상관없이 항상 목록 최상단에 고정합니다.
  * ✎ **수정:** 등록된 할 일 내용과 날짜를 수정합니다.
  * ▼/▲ **하위 할 일:** 세부적인 하위 할 일을 추가하거나 접어둘 수 있습니다.
  * ✕ **삭제:** 할 일을 삭제합니다.
* **설정 저장:** 작성한 할 일 목록은 `settings.json` 파일에 실시간으로 저장되어, 프로그램을 껐다 켜도 데이터가 날아가지 않습니다.

To-Do 리스트가 아닌 읽을 책 리스트, 장보기 리스트 등으로 자유롭게 제목을 수정하여 사용해주시면 됩니다.
